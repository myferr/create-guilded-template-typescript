## 📁 File structure

Inside of your TouchGuild project, you'll see the following folders and files:

```
/
└── commands/
└── events/
└── handlers/
└── config/
|   └── config.json
|
└── index.js
└── package.json
└── README.md
```

Inside of your `config/config.json` you can enter your bot token and prefix.
